#Warning! {backend/}
be sure you are in a correct directory inside your terminal and venv is active.

--- .env guide
create an .env file in backend root directory, ( variable name should be "KEY" )
KEY=input_secret_key_here_on_.env_file
run this command to generate a key and set that to KEY:
python -c "from django.core.management.utils import get_random_secret_key; print(get_random_secret_key())"


--- additional for venv usages
python -m venv {foldername}
source venv/scripts/activate (bash)
pip install -r __requirements.txt

--- Run it after cloning for the first Time!
python manage.py makemigrations
python manage.py migrate
python manage.py collectstatic

--- for creating an admin user
python manage.py createsuperuser

--- running localhost
python manage.py runserver

--- to access django database
http://localhost:8000/admin
http://127.0.0.1:8000/admin
make sure it's on HTTP ! it won't access through HTTPS redirect...

backend localhost always needs to be running ! don't panic if there's 404 not found page
